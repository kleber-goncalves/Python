# Dados, Python e Estatística: Caminhos para STEM

Durante o projeto adquiri conhecimento introdutório em dados, Python e estatística com um projeto final para demonstrar o que aprendi.

Sobre:
A idealização desse projeto foi feita pela, Ana Júlia Amaro, um projeto pessoal e voluntário.
como uma forma de incentivar os jovens a seguirem carreira em STEM uma introdução a dados, Python e estatística durante 3 semanas no mês de janeiro.

Ana Júlia Amaro Pereira Rocha, graduanda em Ciência de Dados e Inteligência Artificial na Escola de Matemática Aplicada da Fundação Getúlio Vargas (FGV-EMAp).
